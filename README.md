# 水果在线购 (FruitShop Online)

## 🛠 技术栈
- Frontend: Vue 3 + Tailwind CSS
- Backend: Spring MVC + MyBatis
- DB: MySQL 8.0

## 🚀 快速启动 (Docker)
1. 确保 Docker Desktop 已运行。
2. 在根目录执行: `docker compose up --build`
3. 访问前端: http://localhost:3000
4. 访问后端: http://localhost:8080/api
